6.2 Model Syntax For MOS Capacitor
----------------------------------

**Input Parameters:**

- w: Top plate(poly) design width [m].

- l: Top plate(poly) design length [m].

**General Syntax:**

Xxx  node_1  node_2  moscap_model_name  w =  l=

**Examples:**

Xc1 1 2 cap_nmos_03v3 w=50u l=50u
